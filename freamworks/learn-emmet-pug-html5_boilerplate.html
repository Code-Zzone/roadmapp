دورة تعلم HTML5 Boilerplate، Pug و Emmet

1. HTML5 Boilerplate

  • HTML5 Boilerplate: هو قالب أساسي لبداية أي مشروع باستخدام HTML5. يشمل إعدادات أساسية ومتوافقة مع معظم المتصفحات.
  • المميزات الرئيسية:
    • دعم كامل لـ **HTML5** و **CSS3** و **JavaScript**.
    • إعدادات مبدئية تساعد في تطوير موقع سريع وآمن.
    • دعم للمتصفحات القديمة والجديدة.
  • الاستخدام: يُستخدم عند بدء أي مشروع ويب ليمنحك هيكلًا نظيفًا ومتوافقًا مع المعايير الحديثة.

2. Pug

  • Pug: هو محرك قوالب يساعدك في كتابة HTML بطريقة مختصرة ونظيفة، عن طريق إزالة الحواجز الزائدة مثل الأقواس و علامات الإغلاق.
  • المميزات الرئيسية:
    • يقلل الكتابة الزائدة بفضل التوجه نحو المسافات.
    • يسمح بتخصيص أكثر ومرونة في تصميم الهيكل العام للصفحة.
    • يستخدم بشكل رئيسي في تطبيقات Node.js لتسهيل بناء واجهات HTML الديناميكية.
  • الاستخدام: عندما ترغب في كتابة HTML سريع ومرن مع تقليل الكود الزائد.

3. Emmet

  • Emmet: هو أداة توفر اختصارات لإنشاء أكواد HTML و CSS بسرعة، من خلال كتابة مختصرات يتم تحويلها إلى أكواد كاملة تلقائيًا.
  • المميزات الرئيسية:
    • يسمح بكتابة أكواد HTML و CSS أسرع باستخدام اختصارات مثل div.container>ul>li*5 والتي تُترجم إلى HTML كامل.
    • يتكامل مع العديد من محررات النصوص مثل **Visual Studio Code** و **Sublime Text**.
    • يدعم تحويل مختصرات CSS إلى خصائص جاهزة باستخدام اختصارات.
  • الاستخدام: يستخدم بشكل رئيسي لتسريع الكتابة وتقليل التكرار في كتابة الأكواد.
