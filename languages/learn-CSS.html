دورة تعلم CSS

1. مقدمة في CSS

  • ما هو CSS؟
  • أهمية CSS في تصميم المواقع.
  • الفرق بين HTML و CSS.
  • كيفية ربط CSS بـ HTML: عبر <link> أو <style>.

2. القواعد الأساسية في CSS

  • قواعد CSS (CSS Rules): مثل selector, property, value.
  • أنواع الـ Selectors: مثل element selector, class selector, id selector, attribute selector.
  • التسلسل في CSS: أهمية ترتيب القواعد والعناصر.
  • استخدام الـ !important (بحذر): لتحديد الأولوية.

3. خصائص النصوص (Text Properties)

  • الخطوط (Fonts): استخدام font-family, font-size, font-weight, font-style.
  • التنسيق (Text Alignment): مثل text-align, line-height.
  • التأثيرات على النصوص: مثل text-transform, letter-spacing, text-decoration.
  • الألوان (Color): استخدام color و background-color.

4. تخطيط الصفحة (Layout)

  • المسافات (Spacing): مثل margin, padding.
  • أبعاد العناصر (Sizing): مثل width, height, max-width, min-height.
  • الحدود (Borders): مثل border, border-radius.
  • الخلفيات (Backgrounds): مثل background-image, background-size, background-position.

5. التصميم باستخدام Flexbox و Grid

  • Flexbox: استخدام display: flex, justify-content, align-items, flex-direction.
  • Grid: استخدام display: grid, grid-template-columns, grid-template-rows, grid-gap.
  • التنسيق المتجاوب (Responsive Design): استخدام وحدات مرنة مثل %, vw, vh, em, و rem.

6. الانتقالات والتأثيرات (Transitions and Animations)

  • الانتقالات (Transitions): استخدام transition, transition-duration, transition-property.
  • التأثيرات (Transformations): مثل transform: rotate, translate, scale.
  • الأنيميشن (Animations): استخدام @keyframes, animation, animation-timing-function.

7. التصميم المتجاوب (Responsive Design)

  • استخدام Media Queries: مثل @media لتحديد قواعد CSS بناءً على حجم الشاشة.
  • المرونة في التصميم: استخدام وحدات مرنة مثل flexbox و grid.
  • إخفاء وإظهار العناصر: مثل display: none و visibility: hidden.

8. أفضل الممارسات في CSS

  • استخدام الأسماء المناسبة في الـ Classes و IDs.
  • تقسيم الشيفرة (Code Splitting): استخدام ملفات CSS منفصلة.
  • إعادة الاستخدام: استخدام CSS Variables مثل --main-color.
  • التحقق من الكود (CSS Validation): استخدام أدوات مثل [CSS Validator](https://jigsaw.w3.org/css-validator/).

9. الأدوات والإطارات (Tools and Frameworks)

  • أدوات تحرير الكود: مثل Visual Studio Code, Sublime Text, أو Atom.
  • إطارات العمل (CSS Frameworks): مثل Bootstrap, Tailwind CSS.
  • Preprocessors: مثل Sass و LESS لتبسيط كتابة CSS.

10. التكامل مع HTML و JavaScript

  • دمج CSS مع HTML: ربط الأنماط بـ HTML باستخدام <link> أو <style>.
  • استخدام JavaScript للتفاعل مع CSS: تغيير الأنماط باستخدام document.style أو classList.
  • إضافة تأثيرات باستخدام CSS و JavaScript معًا.

11. التعلم المستمر ومشاركة المجتمع

  • متابعة المصادر المتجددة: مثل MDN Web Docs، وCSS-Tricks.
  • مشاركة العمل في مجتمعات البرمجة: مثل GitHub، Stack Overflow، Reddit.
  • إعداد مشاريع تطبيقية: مثل بناء صفحات ومواقع كاملة باستخدام CSS.
