دورة تعلم JavaScript

1. مقدمة في JavaScript

  • ما هو JavaScript؟: لغة البرمجة التي تستخدم لإضافة تفاعلات على صفحات الويب.
  • التكامل بين JavaScript و HTML: كيفية استخدام JavaScript في تحسين تجربة المستخدم على صفحات الويب.
  • المتصفح و JavaScript: كيف يقوم المتصفح بتنفيذ أكواد JavaScript.

2. الأساسيات في JavaScript

  • المتغيرات (Variables): استخدام var, let, و const لتخزين البيانات.
  • أنواع البيانات (Data Types): مثل string, number, boolean, array, object, null, undefined.
  • العمليات الحسابية (Operators): مثل +, -, *, /, %.
  • التعليقات (Comments): كيفية إضافة تعليقات في الكود باستخدام // و /* */.

3. الهياكل الشرطية (Conditional Statements)

  • عبارة if: التحقق من شرط باستخدام if و else.
  • التفرعات (Switch): استخدام switch لاختيار بين عدة خيارات.
  • المقارنات (Comparisons): مثل ==, ===, !=, >, <.

4. الحلقات (Loops)

  • حلقة for: استخدام for لتكرار العمليات على مجموعة من البيانات.
  • حلقة while: استخدام while لتكرار العمليات طالما أن الشرط صحيح.
  • حلقة do-while: استخدام do-while لتنفيذ الكود مرة واحدة على الأقل ثم التحقق من الشرط.

5. الدوال (Functions)

  • تعريف الدوال: كيفية إنشاء دالة باستخدام function.
  • تمرير المعاملات (Parameters): كيفية استخدام المعاملات في الدوال.
  • إرجاع القيم (Return): استخدام return لإرجاع قيمة من الدالة.
  • الدوال المجهولة (Anonymous Functions): تعريف الدوال بدون أسماء.

6. المصفوفات (Arrays)

  • تعريف المصفوفات: كيفية إنشاء مصفوفة باستخدام [].
  • الوصول للعناصر في المصفوفة: استخدام الفهرس للوصول إلى العناصر.
  • دوال المصفوفات: مثل push(), pop(), shift(), unshift(), forEach(), map().

7. الكائنات (Objects)

  • تعريف الكائنات: إنشاء كائن باستخدام {}.
  • الوصول للخصائص والقيم: استخدام النقطة . أو الأقواس المربعة [].
  • الدوال في الكائنات: كيفية إضافة وظائف إلى الكائنات.

8. التعامل مع DOM (Document Object Model)

  • الوصول للعناصر في DOM: استخدام getElementById(), getElementsByClassName(), querySelector().
  • تغيير محتوى العناصر: استخدام innerHTML, textContent, value.
  • إضافة أو إزالة العناصر: مثل appendChild(), removeChild().

9. الأحداث (Events)

  • التعامل مع الأحداث: مثل click, mouseover, keyup.
  • إضافة مستمعي الأحداث (Event Listeners): مثل addEventListener().
  • إلغاء الأحداث: استخدام removeEventListener().

10. البرمجة غير المتزامنة (Asynchronous Programming)

  • الموعد الزمني (Timeout): استخدام setTimeout() لتنفيذ دالة بعد فترة زمنية معينة.
  • التكرار الزمني (Interval): استخدام setInterval() لتنفيذ دالة بشكل دوري.
  • التعامل مع الـ Promises: كيفية استخدام Promise مع then() و catch().
  • التعامل مع الـ async/await: طريقة كتابة البرمجة غير المتزامنة بشكل أكثر وضوحًا.

11. أدوات JavaScript المتقدمة

  • استخدام JSON: كيفية استخدام JSON.parse() و JSON.stringify() لتحويل البيانات بين JavaScript و JSON.
  • المصفوفات الدلالية (Map, Set): كيفية استخدام Map و Set للتخزين المسبق للمجموعات.

12. أدوات الاختبار والتطوير

  • استخدام أدوات المتصفح: مثل أدوات المطور في Chrome أو Firefox لتصحيح الأخطاء.
  • اختبار JavaScript: استخدام أدوات مثل Jest أو Mocha لاختبار الأكواد.

13. إعداد دورات تدريبية في JavaScript

  • كتابة شروحات تعليمية: تعلم كيفية شرح المفاهيم بشكل بسيط وواضح.
  • التفاعل مع المتعلمين: عبر إنشاء اختبارات، مشاريع، وأمثلة تفاعلية.
